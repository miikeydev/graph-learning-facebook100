### Dataset

Le dataset Facebook100 contient les réseaux d'amitié de 100 universités américaines capturés à l'automne 2005. Chaque fichier `.gml` dans le répertoire `data/` représente le réseau d'une université avec les attributs des noeuds incluant le statut étudiant/enseignant, le genre, la filière, le dortoir et l'année de diplôme.
